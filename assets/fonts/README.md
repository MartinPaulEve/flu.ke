| File | Font | Source | Licence |
|---|---|---|---|
| `bricolage-grotesque.woff2` | Bricolage Grotesque | https://github.com/ateliertriay/bricolage | OFL 1.1 — `bricolage-grotesque-OFL.txt` |
| `spline-sans-mono.woff2` | Spline Sans Mono | https://github.com/SorkinType/SplineSansMono | OFL 1.1 — `spline-sans-mono-OFL.txt` |
| `og-display.ttf` | Bricolage Grotesque (static instance) | derived from the above | OFL 1.1 — `bricolage-grotesque-OFL.txt` |

`og-display.ttf` is a static instance of Bricolage Grotesque, produced for the
Pillow Open Graph image renderer (`apps/blog/og.py`), which needs a non-variable
TTF. As a derivative of an OFL font it remains under the OFL 1.1.
